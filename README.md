## Unicode Codepoints

Latin-1 is in the usual range 32--255.

Graphic tiles are codepoints 256--383 (U+0100 to U+017F).

The symbols (arrows, Aries, Omega, etc.) are codepoints 384--411 (U+0180 to U+019B).
